Αντικειμενοστραφές παιχνίδι κονσόλας όπου Vampires και Werewolves μάχονται σε έναν δυναμικά δημιουργημένο κόσμο. 
Ο παίκτης επιλέγει πλευρά, κινεί τον avatar του στον χάρτη, αλληλεπιδρά με τις οντότητες, και χρησιμοποιεί μαγικά φίλτρα για να υποστηρίξει την ομάδα του. 
Περιλαμβάνει τυχαία κίνηση, σύστημα μάχης, healing, και εναλλαγή ημέρας-νύχτας.
